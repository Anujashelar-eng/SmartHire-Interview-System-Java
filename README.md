# SmartHire-Interview-System-Java
A Java-based interview preparation system using JavaFX, Firebase, and REST APIs for interactive learning and practice.

## Tech Stack

* Core Java
* JavaFX
* Firebase
* REST API

## Features
- Topic-wise aptitude notes  
- Computer fundamentals notes  
- Topic-wise tests for practice  
- Company-wise interview questions  
- Full-length mock tests  
- Mock interview simulation  
- Resume builder  
- Feedback management system
